ClassFromMapExtension extension Null safety

This converts a Map<String, dynamic> to the given class For example

class Test{
  String name = 'name';
  int age = 0;
}

{
  "name": 'My name',
  "age": 12
}.toClass<Test>();

to

Test{
  name = 'My name';
  age = 12;
}
on
  • Map<String, dynamic>
